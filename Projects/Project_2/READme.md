# Project 2: Full Stack Todo App

## Overview
This project is a full stack backend application that implements authentication and data persistence.

It is designed to simulate a real-world backend system with user authentication and task management.

---

## Features
- User registration and login
- Password hashing using bcrypt
- JWT-based authentication
- SQLite database integration

---

## Technologies Used
- Node.js
- Express
- SQLite
- bcryptjs
- jsonwebtoken

---

## Project Structure
src/
  server.js → Main server file
  routes/ → API routes
  controllers/ → Business logic
  models/ → Database logic

Public/ → Frontend files 

## How to run
npm install
npm run dev
